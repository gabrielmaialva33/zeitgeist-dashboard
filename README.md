# Zeitgeist Dashboard

Intelligence workbench for the [Zeitgeist](https://github.com/gabrielmaialva33/zeitgeist) platform.

## Quick Start

```bash
npm install
npm run dev    # http://localhost:3000
```

Requires Zeitgeist backend running on `http://localhost:4000`.

## Stack

- Next.js 16 + React 19
- Metronic v9.4.6 (Tailwind 4)
- deck.gl + maplibre (maps)
- TanStack Query (data)
- WebSocket (real-time)
